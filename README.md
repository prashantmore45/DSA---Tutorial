## How to Use

```bash
# clone
git clone https://github.com/prashantmore45/DSA---Tutorial.git
cd DSA---Tutorial

# open any topic folder and run a file
g++ <file>.cpp -o app
./app
```

> Some folders may contain multiple files; compile the file you want to run.
